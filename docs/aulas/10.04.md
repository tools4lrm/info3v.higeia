
# Guia de Configuração de Ambiente e Projeto

### 1. Verificação e Instalação do Ambiente

Antes de começar, certifique-se de que as ferramentas básicas estão instaladas e configuradas no seu terminal. Caso não, devemos baixar o arquivo **Ambiente Git / Maven** do 
Materiais de Aula e descompactá-lo na partição C do seu computador.  

Uma vez descompactado, você deverá **executar** o arquivo `C:\dev\tools\Iniciar_Ambiente.bat` que abrirá um ambiente de linha de comando. Nele, digite `mvn -version` para 
verificar a versão do Maven e do JDK (ex: JDK 21). Depois, digite `git -v` para garantir que o Git está pronto para uso.

### 2. Clonagem do Repositório

Navegue até a pasta onde deverá ser salvo os seus projetos (ex: C:\dev) e siga os passos:

No GitHub, clique no botão `<> Code` e copie a URL do repositório.

<img width="1089" height="630" alt="image" src="https://github.com/user-attachments/assets/85b300e7-cd93-4b1d-81aa-bf97c60313f7" />

No terminal, execute o comando `git clone ...` para copiar o seu repositório do github para sua máquina local :

```Bash

git clone https://github.com/usuario/nome-do-repositorio.git
```

Copiado para a sua máquina local, entre no seu repositório.
```Bash

cd nome-do-repositorio
```
Por fim, use `git status` para confirmar que você está no branch principal (main).

<img width="878" height="469" alt="image" src="https://github.com/user-attachments/assets/158e5836-83ce-42f2-b7cd-6c9c2ba060e6" />

### 3. Configuração de Identidade do Git

Caso seja seu primeiro acesso, configure seu nome e e-mail para que seus "commits" sejam identificados:

```Bash

git config user.name "Seu Nome"
git config user.email "seu-email@exemplo.com"
```

### 4. Configuração no Visual Studio Code (VS Code)

Para que o ambiente funcione corretamente dentro do editor, siga estas etapas de customização:

**Instalação de Extensões**
1. Abra o menu de Extensões (`Ctrl+Shift+X`).
2. Pesquise e instale o Extension Pack for Java (da Microsoft).

<img width="1150" height="404" alt="image" src="https://github.com/user-attachments/assets/5e415a67-74a4-431b-a739-dc57c6164fca" />

**Ajuste de Caminhos (Settings)**

1. Acesse as configurações (`Ctrl + ,`) e defina os caminhos, se necessário :
2. Maven: Procure por `maven.executable.path` e insira o caminho do executável (ex: `C:\dev\tools\apache-maven-3.9.14\bin\mvn.cmd`).
3. Git: Procure por git.path para garantir que o VS Code localize o `git.exe` no diretório `C:\dev\tools\PortableGit\bin`.

**Abrindo o Projeto**

1. Vá em File > Open Folder.
2. Navegue até o diretório onde você clonou o projeto (ex: `C:\dev\info3v.higeia` ).
3. Clique em Selecionar Pasta.
