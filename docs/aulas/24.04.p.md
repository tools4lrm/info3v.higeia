# Ajustes da Configuração de Ambiente e Clonando o Projeto

### 1. Estrutura de Pastas e Artefatos

<img width="3016" height="3180" alt="image" src="https://github.com/user-attachments/assets/b7a520f2-8ce6-4497-b0d1-4bf813d441f9" />

Ao clicar na pasta `dev`, você deverá encontrar as seguintes pastas: 

1. `C:\dev\tools\apache-maven-3.9.14` pasta da ferramenta **Apache Maven**.
2. `C:\dev\tools\PortableGit` pasta da ferramenta **Git Portable**.


----
### 2. Configuração do VS Code para integrar com as ferramentas

<img width="3380" height="2228" alt="image" src="https://github.com/user-attachments/assets/5390ea8b-7566-4fd4-8645-41640e82bd9b" />

``` JSON
{
    "maven.executable.path": "C:\\dev\\tools\\apache-maven-3.9.14\\bin\\mvn.cmd",
    "git.path": "C:\\dev\\tools\\PortableGit\\bin\\git.exe",

    "terminal.integrated.profiles.windows": {
        "Git Bash Portable": {
            "path": "C:\\dev\\tools\\PortableGit\\bin\\bash.exe",
            "icon": "terminal-bash"
        }
    },
    "terminal.integrated.defaultProfile.windows": "Git Bash Portable",

    "terminal.integrated.env.windows": {
        "PATH": "C:\\dev\\tools\\apache-maven-3.9.14\\bin;${env:PATH}",
        "MAVEN_HOME": "C:\\dev\\tools\\apache-maven-3.9.14"
    }

}
```
Caso a configuração esteja tudo ok, você deverá ter a opção de realizar com clone, como na imagem.
